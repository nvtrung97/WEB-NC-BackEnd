# WEB-NC-BackEnd
## Installation
 require **Node.js >= 10** to execute.
1. Clone the project.
WEB-NC-BackEnd
```
git clone 
```
2. Install the dependencies.
```
cd WEB-NC-BackEnd
npm install 
```

3. Start:
```
Develop: npm run dev
```
```
Production: npm start
```
```
Testing: npm start:test
```
