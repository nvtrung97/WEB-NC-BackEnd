
## API Reference

#### Get all category

```http
  GET /api/v1/category
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
|`_id`|`int`| `identity` |
|`name`|`string`|`category name`|

#### Get category

```http
  GET /api/v1/category/${_id}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
|`_id`|`int`| `identity` |
|`name`|`string`|`category name`|
